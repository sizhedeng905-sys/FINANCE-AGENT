- B：仅 `refund/correction` 专用模板允许财务录入负数，其他模板拒绝；更正规则同 A。
- C：按模板和角色配置负数白名单，请同时提供模板与角色清单。

负责人答案：
未回答默认：A，拒绝负数和 confirmed 直接覆盖。

- B：所有成本/支出模板都至少需要一份附件。
